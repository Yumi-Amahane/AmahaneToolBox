# AmahaneToolBox  
ツールボックス  
pip install git+https://github.com/Yumi-Amahane/AmahaneToolBox  
  
matplotlibでフェーザ表示(複素平面表示)する  
PR_Phasor(Re,Im,point,Color)  
Re, Im にそれぞれ実/虚部の配列を入力  
ReにComplex型配列を加えた際はImは不要  
  
  
Shuffle(list)  
リストのシャッフルのリターン  
  
  
matT(A,B)  
二次元配列A,Bの掛け算をリターン。  
AとBは同じサイズの正方行列
